# GRA in Literature

## Plot Development

Narrative structure as hierarchical foam:
- $k=0$: Sentences, scenes
- $k=1": Chapters, episodes (rising/falling action)
- $k=2$: Arcs, themes, character development
- $k=3$: Genre conventions, meta-commentary, meaning

**Foam**: Plot holes, inconsistent characterisation, thematic contradictions
**Nullification**: Foreshadowing, callback, thematic resolution

## Poetry Generation

Use Ramanujan priors (intuitive pattern recognition) with GRA:
1. Capture semantic/emotional seed
2. Expand into hierarchical structure (stanza → line → word)
3. Compute linguistic $\Phi$ (rhyme, meter, meaning coherence)
4. Nullify via revision until stable poem emerges

## Storytelling as Therapy

Narrative therapy via GRA:
- Client's life story = high-foam hierarchy
- Therapist helps identify and nullify contradictions
- Re-authored narrative = stable, coherent hierarchy
