# Empirical Study Design

## Research Question

Does daily nullification practice (via Angel-in-Pocket) reduce cognitive foam and improve psychological well-being?

## Hypotheses

- **H1**: Participants using nullification show greater reduction in $\Phi(k)$ over 4 weeks than control group.
- **H2**: $R_{	ext{hier}}$ correlates negatively with BDI (Beck Depression Inventory) scores.
- **H3**: Nullification practice improves STAI (State-Trait Anxiety Inventory) scores.

## Methodology

### Participants
- $N = 200$ adults (18-65 years)
- Randomised into intervention (nullification) and control (journaling only) groups

### Instruments
1. **Angel-in-Pocket app**: Daily thought capture, emotional ratings, automated $\Phi(k)$ computation
2. **BDI-II**: Pre/post and weekly
3. **STAI**: Pre/post and weekly
4. **PSQI**: Sleep quality (secondary measure)

### Protocol
- **Week 0**: Baseline assessment
- **Weeks 1-4**: Daily app usage (intervention) or standard journaling (control)
- **Week 5**: Post-assessment
- **Week 12**: Follow-up

### Data Analysis
- Mixed-effects models for repeated measures
- Correlation analysis: $\Phi(k)$ vs BDI/STAI
- Machine learning: predict $R_{	ext{hier}}$ from thought patterns

## Ethics
- IRB approval required
- Informed consent
- Data anonymisation
- Right to withdraw

## Expected Outcomes
- Publication in open-access journal
- Open dataset (anonymised)
- Updated nullification algorithms
