# Angel-in-Pocket Workflow

## Getting Started

### 1. Installation
- iOS: App Store → "Angel-in-Pocket"
- Android: Google Play → "Angel-in-Pocket"
- Web: https://angel.gra-manifest.org

### 2. Onboarding
- Create account (email or OAuth)
- Complete initial assessment (5 min)
- Set notification preferences
- Grant microphone permission (optional, for voice capture)

### 3. First Capture
- Tap the "+" button
- Type or speak your first thought
- Slide the emotion bar
- Tap "Save"

## Core Workflows

### Workflow A: Quick Capture
**Trigger**: Random thought or emotion arises
**Steps**:
1. Open app (or use widget)
2. Voice or text input
3. Auto-detected level (editable)
4. Save (2 seconds)

### Workflow B: Structured Reflection
**Trigger**: Scheduled evening reflection
**Steps**:
1. Navigate to "Reflect" tab
2. Answer 3 guided questions:
   - "What foam did you notice today?"
   - "Which level was most active?"
   - "What will you nullify tomorrow?"
3. Review auto-generated insights
4. Set tomorrow's intention

### Workflow C: Crisis Nullification
**Trigger**: High emotional distress
**Steps**:
1. Open app → "SOS" button
2. Guided breathing exercise (30 seconds)
3. Rapid capture of distressing thoughts
4. Suggested nullification strategies
5. Option to share with trusted contact

### Workflow D: Weekly Planning
**Trigger**: Sunday evening
**Steps**:
1. Review weekly report
2. Identify top 3 foam sources
3. Create nullification goals for the week
4. Schedule specific practice times
5. Share with accountability partner (optional)

## Advanced Features

### Semantic Search
Search your thought history by meaning, not just keywords:
- "Show me thoughts about feeling stuck"
- "Find patterns around work anxiety"

### What-If Simulator
Test nullification strategies before applying:
- Adjust hierarchy weights
- Simulate different strategies
- Preview expected outcomes

### Export
- PDF report for therapy sessions
- CSV for personal data analysis
- JSON for developers

## Integration

### Calendar
- Auto-detect stressful events
- Pre-emptive nullification reminders
- Post-event reflection prompts

### Wearables
- Heart rate variability → foam proxy
- Sleep data → stability correlation
- Activity → capture triggers

### LLM Assistants
- Chat with your personal GRA coach
- Ask: "Why is my foam high today?"
- Get personalised nullification suggestions

## Privacy

- All data encrypted at rest (AES-256)
- End-to-end encryption for sync
- Local processing for embeddings
- Optional anonymous research contribution
- Full data deletion on request
