# Daily Nullification Guide

## Morning Routine (15 min)

### 1. Thought Capture (5 min)
Open Angel-in-Pocket immediately after waking. Record:
- First 3 thoughts (no filtering)
- Dominant emotion (-1 to +1)
- Any physical sensations

### 2. Quick Foam Scan (5 min)
Review yesterday's clusters. Ask:
- Which cluster had the highest $\Phi$?
- What level was it? (action, emotion, value)
- Is there a recurring pattern?

### 3. Intention Setting (5 min)
Set 1-2 nullification intentions for the day:
- "I will notice when [specific pattern] arises"
- "When I feel [emotion], I will [nullification strategy]"

## Daytime Practice (Ongoing)

### Capture Triggers
- Set 3 random reminders (e.g., 10:00, 14:00, 18:00)
- At each trigger: pause, breathe, capture current thought/emotion
- Tag with context (work, home, commute, etc.)

### Real-Time Nullification
When you notice foam building:
1. **Label**: "This is level-X foam about [topic]"
2. **Breathe**: 3 deep breaths (wave-like pause)
3. **Choose**: Soft nullification (reframe) or hard (walk away)
4. **Log**: Brief note in app

## Evening Reflection (20 min)

### 1. Cluster Review (10 min)
- View auto-generated clusters for the day
- Identify the "foam champion" (highest $\Phi$ cluster)
- Write 2-3 sentences about it

### 2. Nullification Log (5 min)
- How many nullifications did you attempt?
- Which strategy worked best?
- Any surprises?

### 3. Tomorrow Preview (5 min)
- Based on today's data, what might trigger foam tomorrow?
- Set a proactive nullification plan

## Weekly Deep Dive (1 hour)

### 1. Report Review
- Open weekly report in Angel-in-Pocket
- Note trends in $\Phi(k)$ and $R_{	ext{hier}}$
- Compare to previous weeks

### 2. Strategy Adjustment
- If $R_{	ext{hier}}$ is decreasing: add meta-reflection
- If $\Phi$ is rising: increase nullification frequency
- If stable: experiment with new strategies

### 3. Community Share (Optional)
- Share anonymised insights with GRA community
- Learn from others' nullification patterns
- Contribute to collective intelligence

## Monthly Reset

Once per month, perform a "hard nullification":
1. Review all clusters from the month
2. Identify 1-2 deep patterns (usually level 2-3)
3. Write a personal manifesto update
4. Archive old data, start fresh

## Tips for Success

- **Consistency > intensity**: 5 minutes daily beats 1 hour weekly
- **Non-judgmental**: Foam is data, not failure
- **Playful**: Treat it as a game, not a chore
- **Social**: Find a nullification buddy for accountability
